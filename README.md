# Prerequisites

To run this project, you need to have `Node.js` and `npm` installed on your system.

# Setup and run the game

```sh
# Install dependencies
npm install

# Start the project
npm start
```

# Building the game

```sh
# Compile the game into a bundle, which can be found in `dist/`
npm run build
```
